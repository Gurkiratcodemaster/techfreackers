Proposed Solution
Digital Mutual Agreement + Reputation Badge System
HACKMOL 7.0
MoneyLend — A cross-platform (Android + Web) Digital Mutual Agreement platform with a Reputation Badge System
Digital Mutual Agreement (DMA)
› Lender creates agreement in app
› SMS sent to borrower via Fast2SMS
› Borrower signs via app or web
› Legally structured consent flow
Reputation Badge System
› Bronze → Silver → Gold → Platinum
› Tracks every repayment event
› Social proof: '12 loans repaid'
› Dual-confirm updates (both parties)
UPI Repayment & Smart Alerts
› Repay via UPI Deep Link on web
› Smart adaptive reminders
› Missed payment poll flow
› Chain payment notifications
vs Existing Solutions
KhataBook
Expense tracking only. No DMA, no trust, no flow.
Banks / NBFCs
Heavy KYC, no peer-to-peer informal lending.
MoneyLend ✅
DMA + Reputation + SMS flow + UPI + Web fallback.

Key
Features
What MoneyLend does

Features of Solution
Comprehensive Lending Lifecycle Management
HACKMOL 7.0
DMA Signing Flow
App deep link or web fallback. Auto-fetch user info. T&C acceptance. Digital e-sign locks agreement.
Reputation Badges
4-tier badge system tracks repayment history. Social proofs like '12 loans repaid' build trust.
Missed Payment Poll
SMS link opens poll on web/app. No Money / Forgot / Other. Penalty + instalment offer based on badge.
Adaptive Reminders
Reminder timing adjusts to borrower's history. Early for late-payers, near deadline for reliable ones.
UPI Payments & Chain Alerts
Pay via UPI Deep Link from web. Smart chain alert: 'You got ₹X from B — you also owe ₹Y to C.'
Lender Dashboard
Money lent, recovered, at-risk users. 1-tap caller button appears for overdue borrowers.
